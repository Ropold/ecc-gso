# Problem-Solution Examples

| Example | Problem | Solution |
|---------|---------|----------|
| 1 | The delivery is late. | We'll send it by express. You'll get it tomorrow. |
| 2 | We got the wrong color. | Keep it. We'll send the correct one today. |
| 3 | The price on the invoice is wrong. | We'll send a corrected invoice today. |
| 4 | The items arrived broken. | Please return them. We'll refund you immediately. |
| 5 | The manual is missing. | We'll email you the manual today. |
| 6 | We ordered 100, but only got 75. | We'll ship the missing 25 tomorrow. |
| 7 | The payment failed. | Here's the corrected invoice with new bank details. |
| 8 | The delivery will be too late. | We've upgraded to overnight delivery for free. |
| 9 | The product doesn't work. | We'll send a replacement immediately. |
| 10 | The package never arrived. | We'll investigate and send a new one today. |
